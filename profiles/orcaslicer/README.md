# Perfis — OrcaSlicer

Coloque aqui os arquivos `.json` de perfis exportados do OrcaSlicer.

## Como exportar um perfil

1. No OrcaSlicer: **File → Export → Export current configs**
2. Salve o `.json` nesta pasta com um nome descritivo

## Convenção de nomes

```
<material>-<finalidade>-<qualidade>.json

Exemplos:
  pla-padrao-02mm.json
  pla-rapido-03mm.json
  petg-mecanico-02mm.json
  tpu-flexivel-015mm.json
```
