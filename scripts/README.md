# 🔧 Scripts

Scripts utilitários para o dia a dia do repositório.

---

## new-print.sh

Adiciona uma nova entrada no diário de impressões interativamente.

```bash
chmod +x scripts/new-print.sh
./scripts/new-print.sh
```

---

## list-models.sh

Lista todos os modelos (.stl, .3mf, .obj) organizados por categoria, com tamanho de arquivo.

```bash
chmod +x scripts/list-models.sh
./scripts/list-models.sh
```

---

## Tornar todos os scripts executáveis de uma vez

```bash
chmod +x scripts/*.sh
```
